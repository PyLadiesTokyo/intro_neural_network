# Machine Learning Workshop

機械学習初心者向けワークショップ資料

## ハンズオン

`notebooks` 以下のNotebookを使って行う．  
AnacondaのPython 3.5環境が入っている事が前提．  
実行方法は以下の通り．

```bash
$ cd notebooks
$ ipython notebook
```

`01` から順番に実行すること．
